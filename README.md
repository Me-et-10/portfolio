## üí° Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

*   Node.js (for potential build steps if you use a bundler like Webpack or Parcel, or for running scripts)
*   npm or Yarn (package manager)

### Installation

1.  **Clone the repo:**
    ```bash
    git clone https://github.com/Me-et-10/portfolio.git
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd portfolio/portfolio-website # Assuming your main project files are here
    ```
3.  **Install NPM packages:**
    ```bash
    npm install
    # or
    yarn install
    ```

### Usage

To open the portfolio locally, simply open the `index.html` file (or your main HTML file) in your web browser. If you have a local server setup, you can serve the `portfolio-website` directory.
